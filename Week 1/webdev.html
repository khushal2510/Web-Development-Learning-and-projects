What is HTML?

HTML stands for Hyper Text Markup Langauge,but it actually means -

  • Hyper : interconnection/linking of pages(Web pages i mean)
  • Text : textual format
  • Markup : which attributes/tags to define the content
  • Language : set of instructions to be followed for writing/defining
